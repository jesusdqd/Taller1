¿Que es Git?

Es un sistema de control de versiones que te ayuda a gestionar y mantener el historial de cambios en tus archivos y proyectos de software.

Instalación de Git

Para instalar Git en tu computadora, primero visita el sitio web oficial de Git (https://git-scm.com/) y descarga la versión adecuada para tu sistema operativo (Windows, macOS, Linux). Una vez descargado el instalador, ejecútalo y sigue las instrucciones del asistente de instalación. En Windows, puedes optar por usar Git Bash como terminal de Git para acceder a los comandos de Git de manera más familiar. En macOS y Linux, también puedes instalar Git a través de gestores de paquetes como Homebrew en macOS o apt en Ubuntu.

Configuracion inicial

Git trae una herramienta llamada git config, que te permite obtener y establecer variables de configuración que controlan el aspecto y funcionamiento de Git.

¿Que es GitHub?

Es una plataforma en línea donde puedes almacenar y compartir tus proyectos de software utilizando el sistema de control de versiones Git. Permite a los desarrolladores colaborar, revisar y gestionar cambios en el código de manera efectiva.

Crear una cuenta en Github

1. Vaya a https://github.com/. 2. Haga click en registrarse. 3. Sigue las indicaciones para crear tu cuenta personal.

Diferencias y similitudes entre Git y Github.

Las diferencias son que Git es el sistema de control de versiones en sí mismo, que se utiliza localmente en tu máquina y GitHub es una plataforma en línea que proporciona servicios para alojar repositorios Git de forma remota y colaborar en proyectos.

Las similitudes son Ambos están diseñados para gestionar y mantener el historial de cambios en proyectos de software y Utilizan comandos similares para operaciones básicas como commit, branch, merge, etc.
